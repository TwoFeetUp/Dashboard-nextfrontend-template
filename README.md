# Olympisch Stadion AI Assistants

This is a Next.js project for Olympisch Stadion AI Assistants platform.

## Getting Started

First, install the dependencies:

```bash
npm install
# or
yarn install
# or
pnpm install
```

Then, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `src/app/` - App Router pages and layouts
- `src/components/` - React components
- `playground/` - Experimental code and testing

## Tech Stack

- Next.js 14 with App Router
- TypeScript
- Tailwind CSS
- PocketBase (for authentication)
- Vercel AI SDK with Mistral API
- shadcn/ui components
